# RecipeVault

RecipeVault is a modern recipe management application that helps users organize recipes into collections and follow cooking instructions using an immersive step-by-step experience inspired by WhatsApp Status.

---

## ✨ Features

- 🍽 Create and manage recipes
- 📚 Organize recipes into collections
- 📝 Detailed ingredients list
- 📖 Step-by-step cooking instructions
- 📱 Full-screen recipe step viewer (Status-style)
- 🔍 Search recipes instantly
- ❤️ Mark favorite recipes
- 🌙 Light & Dark Theme
- 🌍 Multiple language support
- 💾 Offline-first experience

## 🏗 Architecture

RecipeVault follows Google's recommended MVVM architecture.

```
UI (Jetpack Compose)
        │
    ViewModel
        │
   Repository
        │
 Room Database
```

---

## 🚀 Why I Built RecipeVault

Many recipe applications become cluttered with advertisements or require an internet connection. RecipeVault was built to provide a clean, distraction-free experience where users can organize their personal recipes and cook without interruptions.

The highlight of the app is its immersive recipe mode, allowing users to follow each cooking step one at a time using a full-screen interface inspired by WhatsApp Status. This makes cooking easier by focusing on the current instruction instead of displaying the entire recipe at once.
